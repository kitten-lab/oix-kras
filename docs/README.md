## what is _my pocket internet_?
it is a personal data system that works like a tiny internet  

--  
notes, entities, and events are all connected, tagged, and tracked over time and relativity.  

instead of static notes, everything has a relationship and "narrative gravity"  
meaning you can explore how things evolve, connect, and resurface over a wide array of data.  

it runs on rapidly generated web interfaces with high customization, all powering the same underlying system.
